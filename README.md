# JsonToCsv

JsonToCsv is a high-performance, stateless .NET 10 command-line tool that parses JSON datasets and flattens them into structured CSV outputs. It supports custom delimiter configurations, special whitespace sequence evaluation, strict syntax validation, and atomic filesystem recovery.

## Features
- **Atomic Output Creation**: Guarantees file operations are cleanly rolled back by deleting partial output files if validation, syntax, or rendering errors occur midway.
- **Strict Structural Validation**: Rejects malformed roots, flat arrays of primitives, or elements that cannot be mapped into tabular formats.
- **Comprehensive Syntax Diagnostics**: Parses JSON via native DOM mechanisms, reporting exact line and column positions to `stderr` during syntax issues.
- **High-Performance JSON Flattening**: Recursively flattens complex nested JSON maps into dot-notated tabular structures.
- **Safe Deep-Recursion Protection**: Restricts nested elements to 100 levels, safely raising structured validation errors instead of risking a `StackOverflowException`.
- **Stateless RFC 4180-Compliant Writing**: Supports complex cells (containing commas, delimiters, double quotes, or line breaks) and aligns heterogeneous schemas cleanly with empty columns.
- **Flexible Delimiters**: Configurable custom column delimiters, including standard punctuation (e.g., `,`, `;`) or escaped characters like tab indentation (`\t`).

## Getting Started

### Prerequisites
- .NET 10 SDK (v10.0.100 or higher)

### Installation & Build
```sh
for i in 1 2 3; do dotnet restore --disable-parallel && exit 0; sleep 5; done; exit 1
dotnet build
```

## Usage
Translate a nested JSON array file into a standard comma-separated table:
```sh
dotnet run --project src/JsonToCsv/JsonToCsv.csproj run -i path/to/input.json -o path/to/output.csv -d ","
```

Translate JSON using tab spacing as a column delimiter:
```sh
dotnet run --project src/JsonToCsv/JsonToCsv.csproj run -i path/to/input.json -o path/to/output.csv -d "\t"
```
